 \hypertarget{classnumpy_1_1ma_1_1extras_1_1__fromnxfunction_autotoc_md6}{}\doxysubsubsection{\texorpdfstring{-\/--- --- Standard functions ---}{-\/--- --- Standard functions ---}}\label{classnumpy_1_1ma_1_1extras_1_1__fromnxfunction_autotoc_md6}
\begin{DoxyVerb}Defines a wrapper to adapt NumPy functions to masked arrays.


An instance of `_fromnxfunction` can be called with the same parameters
as the wrapped NumPy function. The docstring of `newfunc` is adapted from
the wrapped function as well, see `getdoc`.

This class should not be used directly. Instead, one of its extensions that
provides support for a specific type of input should be used.

Parameters
----------
funcname : str
    The name of the function to be adapted. The function should be
    in the NumPy namespace (i.e. ``np.funcname``).
\end{DoxyVerb}
