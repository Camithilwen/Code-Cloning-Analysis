\doxysubsection{Detailed Description}
\begin{DoxyVerb}`BboxTransformToMaxOnly` is a transformation that linearly transforms points from
the unit bounding box to a given `Bbox` with a fixed upper left of (0, 0).
\end{DoxyVerb}
