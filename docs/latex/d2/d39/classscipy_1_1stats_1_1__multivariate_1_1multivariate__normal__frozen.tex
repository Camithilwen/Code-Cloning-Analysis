scipy.\+stats.\+\_\+multivariate.\+multivariate\+\_\+normal\+\_\+frozen.\+\_\+\+\_\+init\+\_\+\+\_\+ (\begin{DoxyParamCaption}\item[{}]{self}{, }\item[{}]{mean}{ = {\ttfamily None}, }\item[{}]{cov}{ = {\ttfamily 1}, }\item[{}]{allow\+\_\+singular}{ = {\ttfamily False}, }\item[{}]{seed}{ = {\ttfamily None}, }\item[{}]{maxpts}{ = {\ttfamily None}, }\item[{}]{abseps}{ = {\ttfamily 1e-\/5}, }\item[{}]{releps}{ = {\ttfamily 1e-\/5}}\end{DoxyParamCaption})}

\begin{DoxyVerb}Create a frozen multivariate normal distribution.

Parameters
----------
mean : array_like, default: ``[0]``
    Mean of the distribution.
cov : array_like, default: ``[1]``
    Symmetric positive (semi)definite covariance matrix of the
    distribution.
allow_singular : bool, default: ``False``
    Whether to allow a singular covariance matrix.
seed : {None, int, `numpy.random.Generator`, `numpy.random.RandomState`}, optional
    If `seed` is None (or `np.random`), the `numpy.random.RandomState`
    singleton is used.
    If `seed` is an int, a new ``RandomState`` instance is used,
    seeded with `seed`.
    If `seed` is already a ``Generator`` or ``RandomState`` instance
    then that instance is used.
maxpts : integer, optional
    The maximum number of points to use for integration of the
    cumulative distribution function (default `1000000*dim`)
abseps : float, optional
    Absolute error tolerance for the cumulative distribution function
    (default 1e-5)
releps : float, optional
    Relative error tolerance for the cumulative distribution function
    (default 1e-5)

Examples
--------
When called with the default parameters, this will create a 1D random
variable with mean 0 and covariance 1:

>>> from scipy.stats import multivariate_normal
>>> r = multivariate_normal()
>>> r.mean
array([ 0.])
>>> r.cov
array([[1.]])
\end{DoxyVerb}
