\doxysubsection{Detailed Description}
\begin{DoxyVerb}Termination condition for an iteration. It is terminated if

- |F| < f_rtol*|F_0|, AND
- |F| < f_tol

AND

- |dx| < x_rtol*|x|, AND
- |dx| < x_tol
\end{DoxyVerb}
