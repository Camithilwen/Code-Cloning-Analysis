\begin{DoxyVerb}Accessor object for categorical properties of the Series values.

Parameters
----------
data : Series or CategoricalIndex

Examples
--------
>>> s = pd.Series(list("abbccc")).astype("category")
>>> s
0    a
1    b
2    b
3    c
4    c
5    c
dtype: category
Categories (3, object): ['a', 'b', 'c']

>>> s.cat.categories

>>> s.cat.rename_categories(list("cba"))

1    b
2    b
3    a
4    a
5    a
