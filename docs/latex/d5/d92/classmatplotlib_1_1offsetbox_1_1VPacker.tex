\doxysubsection{Detailed Description}
\begin{DoxyVerb}VPacker packs its children vertically, automatically adjusting their
relative positions at draw time.

.. code-block:: none

   +---------+
   | Child 1 |
   | Child 2 |
   | Child 3 |
   +---------+
\end{DoxyVerb}
