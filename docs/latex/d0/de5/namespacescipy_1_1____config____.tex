----------

    Indicates how to display the config information.

    of the configuration.

Returns

out : {`dict`, `None`}
    If mode is `'dicts'`, a dict is returned, else None

Notes

1. The `'stdout'` mode will give more readable
   output if ``pyyaml`` is installed
