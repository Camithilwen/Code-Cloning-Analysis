\doxysubsection{Detailed Description}
\begin{DoxyVerb}Barrier optimization problem:
    minimize fun(x) - barrier_parameter*sum(log(s))
    subject to: constr_eq(x)     = 0
              constr_ineq(x) + s = 0
\end{DoxyVerb}

\begin{DoxyVerb}Assemble sparse Jacobian given its components.

Given ``J_eq``, ``J_ineq`` and ``s`` returns:
    jacobian = [ J_eq,     0     ]
               [ J_ineq, diag(s) ]

It is equivalent to:
    sps.bmat([[ J_eq,   None    ],
              [ J_ineq, diag(s) ]], "csr")
but significantly more efficient for this
given structure.
\end{DoxyVerb}
