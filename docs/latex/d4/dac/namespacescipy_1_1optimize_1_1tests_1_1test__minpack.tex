scipy.\+optimize.\+tests.\+test\+\_\+minpack.\+pressure\+\_\+network (\begin{DoxyParamCaption}\item[{}]{flow\+\_\+rates}{, }\item[{}]{Qtot}{, }\item[{}]{k}{}\end{DoxyParamCaption})}

\begin{DoxyVerb}Evaluate non-linear equation system representing
the pressures and flows in a system of n parallel pipes::

    f_i = P_i - P_0, for i = 1..n
    f_0 = sum(Q_i) - Qtot

where Q_i is the flow rate in pipe i and P_i the pressure in that pipe.
Pressure is modeled as a P=kQ**2 where k is a valve coefficient and
Q is the flow rate.

Parameters
----------
flow_rates : float
    A 1-D array of n flow rates [kg/s].
k : float
    A 1-D array of n valve coefficients [1/kg m].
Qtot : float
    A scalar, the total input flow rate [kg/s].

Returns
-------
F : float
    A 1-D array, F[i] == f_i.
\end{DoxyVerb}
