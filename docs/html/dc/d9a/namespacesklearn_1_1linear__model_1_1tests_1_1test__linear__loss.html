sklearn.linear_model.tests.test_linear_loss.test_multinomial_hessian_3_classes ( sample_weight,
global_random_seed ) 
Test multinomial hessian for 3 classes and 2 points.

For n_classes = 3 and n_samples = 2, we have
  p0 = [p0_0, p0_1]
  p1 = [p1_0, p1_1]
  p2 = [p2_0, p2_1]
and with 2 x 2 diagonal subblocks
  H = [p0 * (1-p0),    -p0 * p1,    -p0 * p2]
      [   -p0 * p1, p1 * (1-p1),    -p1 * p2]
      [   -p0 * p2,    -p1 * p2, p2 * (1-p2)]
  hess = X' H X
