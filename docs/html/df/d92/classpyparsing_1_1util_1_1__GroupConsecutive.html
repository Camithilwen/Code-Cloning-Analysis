Detailed Description

Used as a callable `key` for itertools.groupby to group
characters that are consecutive:
    itertools.groupby("abcdejkmpqrs", key=IsConsecutive())
    yields:
        (0, iter(['a', 'b', 'c', 'd', 'e']))
        (1, iter(['j', 'k']))
        (2, iter(['m']))
        (3, iter(['p', 'q', 'r', 's']))
