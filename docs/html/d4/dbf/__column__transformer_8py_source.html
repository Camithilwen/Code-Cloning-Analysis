63class ColumnTransformer(TransformerMixin, _BaseComposition):
64 """Applies transformers to columns of an array or pandas DataFrame.
65
66 This estimator allows different columns or column subsets of the input
67 to be transformed separately and the features generated by each transformer
68 will be concatenated to form a single feature space.
69 This is useful for heterogeneous or columnar data, to combine several
70 feature extraction mechanisms or transformations into a single transformer.
71
72 Read more in the :ref:`User Guide <column_transformer>`.
73
74 .. versionadded:: 0.20
75
76 Parameters
77 ----------
78 transformers : list of tuples
79 List of (name, transformer, columns) tuples specifying the
80 transformer objects to be applied to subsets of the data.
81
82 name : str
83 Like in Pipeline and FeatureUnion, this allows the transformer and
84 its parameters to be set using ``set_params`` and searched in grid
85 search.
86 transformer : {'drop', 'passthrough'} or estimator
87 Estimator must support :term:`fit` and :term:`transform`.
88 Special-cased strings 'drop' and 'passthrough' are accepted as
89 well, to indicate to drop the columns or to pass them through
90 untransformed, respectively.
91 columns : str, array-like of str, int, array-like of int, \
92 array-like of bool, slice or callable
93 Indexes the data on its second axis. Integers are interpreted as
94 positional columns, while strings can reference DataFrame columns
95 by name. A scalar string or int should be used where
96 ``transformer`` expects X to be a 1d array-like (vector),
97 otherwise a 2d array will be passed to the transformer.
98 A callable is passed the input data `X` and can return any of the
99 above. To select multiple columns by name or dtype, you can use
100 :obj:`make_column_selector`.
101
102 remainder : {'drop', 'passthrough'} or estimator, default='drop'
103 By default, only the specified columns in `transformers` are
104 transformed and combined in the output, and the non-specified
105 columns are dropped. (default of ``'drop'``).
106 By specifying ``remainder='passthrough'``, all remaining columns that
107 were not specified in `transformers`, but present in the data passed
108 to `fit` will be automatically passed through. This subset of columns
109 is concatenated with the output of the transformers. For dataframes,
110 extra columns not seen during `fit` will be excluded from the output
111 of `transform`.
112 By setting ``remainder`` to be an estimator, the remaining
113 non-specified columns will use the ``remainder`` estimator. The
114 estimator must support :term:`fit` and :term:`transform`.
115 Note that using this feature requires that the DataFrame columns
116 input at :term:`fit` and :term:`transform` have identical order.
117
118 sparse_threshold : float, default=0.3
119 If the output of the different transformers contains sparse matrices,
120 these will be stacked as a sparse matrix if the overall density is
121 lower than this value. Use ``sparse_threshold=0`` to always return
122 dense. When the transformed output consists of all dense data, the
123 stacked result will be dense, and this keyword will be ignored.
124
125 n_jobs : int, default=None
126 Number of jobs to run in parallel.
127 ``None`` means 1 unless in a :obj:`joblib.parallel_backend` context.
128 ``-1`` means using all processors. See :term:`Glossary <n_jobs>`
129 for more details.
130
131 transformer_weights : dict, default=None
132 Multiplicative weights for features per transformer. The output of the
133 transformer is multiplied by these weights. Keys are transformer names,
134 values the weights.
135
136 verbose : bool, default=False
137 If True, the time elapsed while fitting each transformer will be
138 printed as it is completed.
139
140 verbose_feature_names_out : bool, str or Callable[[str, str], str], default=True
141
142 - If True, :meth:`ColumnTransformer.get_feature_names_out` will prefix
143 all feature names with the name of the transformer that generated that
144 feature. It is equivalent to setting
145 `verbose_feature_names_out="{transformer_name}__{feature_name}"`.
146 - If False, :meth:`ColumnTransformer.get_feature_names_out` will not
147 prefix any feature names and will error if feature names are not
148 unique.
149 - If ``Callable[[str, str], str]``,
150 :meth:`ColumnTransformer.get_feature_names_out` will rename all the features
151 using the name of the transformer. The first argument of the callable is the
152 transformer name and the second argument is the feature name. The returned
153 string will be the new feature name.
154 - If ``str``, it must be a string ready for formatting. The given string will
155 be formatted using two field names: ``transformer_name`` and ``feature_name``.
156 e.g. ``"{feature_name}__{transformer_name}"``. See :meth:`str.format` method
157 from the standard library for more info.
158
159 .. versionadded:: 1.0
160
161 .. versionchanged:: 1.6
162 `verbose_feature_names_out` can be a callable or a string to be formatted.
163
164 force_int_remainder_cols : bool, default=False
165 This parameter has no effect.
166
167 .. note::
168 If you do not access the list of columns for the remainder columns
169 in the `transformers_` fitted attribute, you do not need to set
170 this parameter.
171
172 .. versionadded:: 1.5
173
174 .. versionchanged:: 1.7
175 The default value for `force_int_remainder_cols` will change from
176 `True` to `False` in version 1.7.
177
178 .. deprecated:: 1.7
179 `force_int_remainder_cols` is deprecated and will be removed in 1.9.
180
181 Attributes
182 ----------
183 transformers_ : list
184 The collection of fitted transformers as tuples of (name,
185 fitted_transformer, column). `fitted_transformer` can be an estimator,
186 or `'drop'`; `'passthrough'` is replaced with an equivalent
187 :class:`~sklearn.preprocessing.FunctionTransformer`. In case there were
188 no columns selected, this will be the unfitted transformer. If there
189 are remaining columns, the final element is a tuple of the form:
190 ('remainder', transformer, remaining_columns) corresponding to the
191 ``remainder`` parameter. If there are remaining columns, then
192 ``len(transformers_)==len(transformers)+1``, otherwise
193 ``len(transformers_)==len(transformers)``.
194
195 .. versionadded:: 1.7
196 The format of the remaining columns now attempts to match that of the other
197 transformers: if all columns were provided as column names (`str`), the
198 remaining columns are stored as column names; if all columns were provided
199 as mask arrays (`bool`), so are the remaining columns; in all other cases
200 the remaining columns are stored as indices (`int`).
201
202 named_transformers_ : :class:`~sklearn.utils.Bunch`
203 Read-only attribute to access any transformer by given name.
204 Keys are transformer names and values are the fitted transformer
205 objects.
206
207 sparse_output_ : bool
208 Boolean flag indicating whether the output of ``transform`` is a
209 sparse matrix or a dense numpy array, which depends on the output
210 of the individual transformers and the `sparse_threshold` keyword.
211
212 output_indices_ : dict
213 A dictionary from each transformer name to a slice, where the slice
214 corresponds to indices in the transformed output. This is useful to
215 inspect which transformer is responsible for which transformed
216 feature(s).
217
218 .. versionadded:: 1.0
219
220 n_features_in_ : int
221 Number of features seen during :term:`fit`. Only defined if the
222 underlying transformers expose such an attribute when fit.
223
224 .. versionadded:: 0.24
225
226 feature_names_in_ : ndarray of shape (`n_features_in_`,)
227 Names of features seen during :term:`fit`. Defined only when `X`
228 has feature names that are all strings.
229
230 .. versionadded:: 1.0
231
232 See Also
233 --------
234 make_column_transformer : Convenience function for
235 combining the outputs of multiple transformer objects applied to
236 column subsets of the original feature space.
237 make_column_selector : Convenience function for selecting
238 columns based on datatype or the columns name with a regex pattern.
239
240 Notes
241 -----
242 The order of the columns in the transformed feature matrix follows the
243 order of how the columns are specified in the `transformers` list.
244 Columns of the original feature matrix that are not specified are
245 dropped from the resulting transformed feature matrix, unless specified
246 in the `passthrough` keyword. Those columns specified with `passthrough`
247 are added at the right to the output of the transformers.
248
249 Examples
250 --------
251 >>> import numpy as np
252 >>> from sklearn.compose import ColumnTransformer
253 >>> from sklearn.preprocessing import Normalizer
254 >>> ct = ColumnTransformer(
255 ... [("norm1", Normalizer(norm='l1'), [0, 1]),
256 ... ("norm2", Normalizer(norm='l1'), slice(2, 4))])
257 >>> X = np.array([[0., 1., 2., 2.],
258 ... [1., 1., 0., 1.]])
259 >>> # Normalizer scales each row of X to unit norm. A separate scaling
260 >>> # is applied for the two first and two last elements of each
261 >>> # row independently.
262 >>> ct.fit_transform(X)
263 array([[0. , 1. , 0.5, 0.5],
264 [0.5, 0.5, 0. , 1. ]])
265
266 :class:`ColumnTransformer` can be configured with a transformer that requires
267 a 1d array by setting the column to a string:
268
269 >>> from sklearn.feature_extraction.text import CountVectorizer
270 >>> from sklearn.preprocessing import MinMaxScaler
271 >>> import pandas as pd # doctest: +SKIP
272 >>> X = pd.DataFrame({
273 ... "documents": ["First item", "second one here", "Is this the last?"],
274 ... "width": [3, 4, 5],
275 ... }) # doctest: +SKIP
276 >>> # "documents" is a string which configures ColumnTransformer to
277 >>> # pass the documents column as a 1d array to the CountVectorizer
278 >>> ct = ColumnTransformer(
279 ... [("text_preprocess", CountVectorizer(), "documents"),
280 ... ("num_preprocess", MinMaxScaler(), ["width"])])
281 >>> X_trans = ct.fit_transform(X) # doctest: +SKIP
282
283 For a more detailed example of usage, see
284 :ref:`sphx_glr_auto_examples_compose_plot_column_transformer_mixed_types.py`.
285 """

433 def _iter(self, fitted, column_as_labels, skip_drop, skip_empty_columns):
434 """
435 Generate (name, trans, columns, weight) tuples.
436
437
438 Parameters
439 ----------
440 fitted : bool
441 If True, use the fitted transformers (``self.transformers_``) to
442 iterate through transformers, else use the transformers passed by
443 the user (``self.transformers``).
444
445 column_as_labels : bool
446 If True, columns are returned as string labels. If False, columns
447 are returned as they were given by the user. This can only be True
448 if the ``ColumnTransformer`` is already fitted.
449
450 skip_drop : bool
451 If True, 'drop' transformers are filtered out.
452
453 skip_empty_columns : bool
454 If True, transformers with empty selected columns are filtered out.
455
456 Yields
457 ------
458 A generator of tuples containing:
459 - name : the name of the transformer
460 - transformer : the transformer object
461 - columns : the columns for that transformer
462 - weight : the weight of the transformer
463 """
464 if fitted:
465 transformers = self.transformers_
466 else:
467 # interleave the validated column specifiers
468 transformers = [
469 (name, trans, column)
470 for (name, trans, _), column in zip(self.transformers, self._columns)
471 ]
472 # add transformer tuple for remainder
473 if self._remainder[2]:
474 transformers = chain(transformers, [self._remainder])
475
476 get_weight = (self.transformer_weights or {}).get
477
478 for name, trans, columns in transformers:
479 if skip_drop and trans == "drop":
480 continue
481 if skip_empty_columns and _is_empty_column_selection(columns):
482 continue
483
484 if column_as_labels:
485 # Convert all columns to using their string labels
486 columns_is_scalar = np.isscalar(columns)
487
488 indices = self._transformer_to_input_indices[name]
489 columns = self.feature_names_in_[indices]
490
491 if columns_is_scalar:
492 # selection is done with one dimension
493 columns = columns[0]
494
495 yield (name, trans, columns, get_weight(name))

714 ----------
715 transformers : list of estimators
716 The fitted estimators as the output of
717 `self._call_func_on_transformers(func=_fit_transform_one, ...)`.
718 That function doesn't include 'drop' or transformers for which no
719 column is selected. 'drop' is kept as is, and for the no-column
720 transformers the unfitted transformer is put in
721 `self.transformers_`.
722 """
723 # transformers are fitted; excludes 'drop' cases
724 fitted_transformers = iter(transformers)
725 transformers_ = []
726
727 for name, old, column, _ in self._iter(
728 fitted=False,
729 column_as_labels=False,
730 skip_drop=False,
731 skip_empty_columns=False,
732 ):
733 if old == "drop":
734 trans = "drop"
735 elif _is_empty_column_selection(column):
736 trans = old
737 else:
738 trans = next(fitted_transformers)
739 transformers_.append((name, trans, column))
740
741 # sanity check that transformers is exhausted
742 assert not list(fitted_transformers)
743 self.transformers_ = transformers_

745 def _validate_output(self, result):
746 """
747 Ensure that the output of each transformer is 2D. Otherwise
748 hstack can raise an error or produce incorrect results.
749 """
750 names = [
751 name
752 for name, _, _, _ in self._iter(
753 fitted=True,
754 column_as_labels=False,
755 skip_drop=True,
756 skip_empty_columns=True,
757 )
758 ]
759 for Xs, name in zip(result, names):
760 if not getattr(Xs, "ndim", 0) == 2 and not hasattr(Xs, "__dataframe__"):
761 raise ValueError(
762 "The output of the '{0}' transformer should be 2D (numpy array, "
763 "scipy sparse array, dataframe).".format(name)
764 )
765 if _get_output_config("transform", self)["dense"] == "pandas":
766 return
767 try:
768 import pandas as pd
769 except ImportError:
770 return

772 if not _is_pandas_df(Xs):
773 continue

776 continue

778 continue
779 class_name = self.__class__.__name__

785 " inefficiencies. To avoid this problem you can (i)"
786 " store the output in a pandas DataFrame by using"
787 f" {class_name}.set_output(transform='pandas') or (ii) modify"
788 f" the input data or the '{name}' transformer to avoid the"
789 " presence of pandas.NA (for example by using"
790 " pandas.DataFrame.astype)."

825 def _call_func_on_transformers(self, X, y, func, column_as_labels, routed_params):
826 """
827 Private function to fit and/or transform on demand.
828
829 Parameters
830 ----------
831 X : {array-like, dataframe} of shape (n_samples, n_features)
832 The data to be used in fit and/or transform.
833
834 y : array-like of shape (n_samples,)
835 Targets.
836
837 func : callable
838 Function to call, which can be _fit_transform_one or
839 _transform_one.
840
841 column_as_labels : bool
842 Used to iterate through transformers. If True, columns are returned
843 as strings. If False, columns are returned as they were given by
844 the user. Can be True only if the ``ColumnTransformer`` is already
845 fitted.
846
847 routed_params : dict
848 The routed parameters as the output from ``process_routing``.
849
850 Returns
851 -------
852 Return value (transformers and/or transformed X data) depends
853 on the passed function.
854 """
855 if func is _fit_transform_one:
856 fitted = False
857 else: # func is _transform_one
858 fitted = True
859
860 transformers = list(
861 self._iter(
862 fitted=fitted,
863 column_as_labels=column_as_labels,
864 skip_drop=True,
865 skip_empty_columns=True,
866 )
867 )
868 try:
869 jobs = []
870 for idx, (name, trans, columns, weight) in enumerate(transformers, start=1):
871 if func is _fit_transform_one:
872 if trans == "passthrough":
873 output_config = _get_output_config("transform", self)
874 trans = FunctionTransformer(
875 accept_sparse=True,
876 check_inverse=False,
877 feature_names_out="one-to-one",
878 ).set_output(transform=output_config["dense"])
879
880 extra_args = dict(
881 message_clsname="ColumnTransformer",
882 message=self._log_message(name, idx, len(transformers)),
883 )
884 else: # func is _transform_one
885 extra_args = {}
886 jobs.append(
887 delayed(func)(
888 transformer=clone(trans) if not fitted else trans,
889 X=_safe_indexing(X, columns, axis=1),
890 y=y,
891 weight=weight,
892 **extra_args,
893 params=routed_params[name],
894 )
895 )
896
897 return Parallel(n_jobs=self.n_jobs)(jobs)
898
899 except ValueError as e:
900 if "Expected 2D array, got 1D array instead" in str(e):
901 raise ValueError(_ERR_MSG_1DCOLUMN) from e
902 else:
903 raise

909 ----------
910 X : {array-like, dataframe} of shape (n_samples, n_features)
911 Input data, of which specified subsets are used to fit the
912 transformers.

914 y : array-like of shape (n_samples,...), default=None
915 Targets for supervised learning.
916

918 Parameters to be passed to the underlying transformers' ``fit`` and

924 .. versionadded:: 1.4
925

928 self : ColumnTransformer
929 This estimator.

941 def fit_transform(self, X, y=None, **params):
942 """Fit all transformers, transform the data and concatenate results.
943
944 Parameters
945 ----------
946 X : {array-like, dataframe} of shape (n_samples, n_features)
947 Input data, of which specified subsets are used to fit the
948 transformers.
949
950 y : array-like of shape (n_samples,), default=None
951 Targets for supervised learning.
952
953 **params : dict, default=None
954 Parameters to be passed to the underlying transformers' ``fit`` and
955 ``transform`` methods.
956
957 You can only pass this if metadata routing is enabled, which you
958 can enable using ``sklearn.set_config(enable_metadata_routing=True)``.
959
960 .. versionadded:: 1.4
961
962 Returns
963 -------
964 X_t : {array-like, sparse matrix} of \
965 shape (n_samples, sum_n_components)
966 Horizontally stacked results of transformers. sum_n_components is the
967 sum of n_components (output dimension) over transformers. If
968 any result is a sparse matrix, everything will be converted to
969 sparse matrices.
970 """
971 _raise_for_params(params, self, "fit_transform")
972 _check_feature_names(self, X, reset=True)
973
974 if self.force_int_remainder_cols != "deprecated":
975 warnings.warn(
976 "The parameter `force_int_remainder_cols` is deprecated and will be "
977 "removed in 1.9. It has no effect. Leave it to its default value to "
978 "avoid this warning.",
979 FutureWarning,
980 )
981
982 X = _check_X(X)
983 # set n_features_in_ attribute
984 _check_n_features(self, X, reset=True)
985 self._validate_transformers()
986 n_samples = _num_samples(X)
987
988 self._validate_column_callables(X)
989 self._validate_remainder(X)
990
991 if _routing_enabled():
992 routed_params = process_routing(self, "fit_transform", **params)
993 else:
994 routed_params = self._get_empty_routing()
995
996 result = self._call_func_on_transformers(
997 X,
998 y,
999 _fit_transform_one,
1000 column_as_labels=False,
1001 routed_params=routed_params,
1002 )
1003
1004 if not result:
1005 self._update_fitted_transformers([])
1006 # All transformers are None
1007 return np.zeros((n_samples, 0))
1008
1009 Xs, transformers = zip(*result)
1010
1011 # determine if concatenated output will be sparse or not
1012 if any(sparse.issparse(X) for X in Xs):
1013 nnz = sum(X.nnz if sparse.issparse(X) else X.size for X in Xs)
1014 total = sum(
1015 X.shape[0] * X.shape[1] if sparse.issparse(X) else X.size for X in Xs
1016 )
1017 density = nnz / total
1018 self.sparse_output_ = density < self.sparse_threshold
1019 else:
1020 self.sparse_output_ = False
1021
1022 self._update_fitted_transformers(transformers)
1023 self._validate_output(Xs)
1024 self._record_output_indices(Xs)
1025
1026 return self._hstack(list(Xs), n_samples=n_samples)

1028 def transform(self, X, **params):
1029 """Transform X separately by each transformer, concatenate results.
1030
1031 Parameters
1032 ----------
1033 X : {array-like, dataframe} of shape (n_samples, n_features)
1034 The data to be transformed by subset.
1035
1036 **params : dict, default=None
1037 Parameters to be passed to the underlying transformers' ``transform``
1038 method.
1039
1040 You can only pass this if metadata routing is enabled, which you
1041 can enable using ``sklearn.set_config(enable_metadata_routing=True)``.
1042
1043 .. versionadded:: 1.4
1044
1045 Returns
1046 -------
1047 X_t : {array-like, sparse matrix} of \
1048 shape (n_samples, sum_n_components)
1049 Horizontally stacked results of transformers. sum_n_components is the
1050 sum of n_components (output dimension) over transformers. If
1051 any result is a sparse matrix, everything will be converted to
1052 sparse matrices.
1053 """
1054 _raise_for_params(params, self, "transform")
1055 check_is_fitted(self)
1056 X = _check_X(X)
1057
1058 # If ColumnTransformer is fit using a dataframe, and now a dataframe is
1059 # passed to be transformed, we select columns by name instead. This
1060 # enables the user to pass X at transform time with extra columns which
1061 # were not present in fit time, and the order of the columns doesn't
1062 # matter.
1063 fit_dataframe_and_transform_dataframe = hasattr(self, "feature_names_in_") and (
1064 _is_pandas_df(X) or hasattr(X, "__dataframe__")
1065 )
1066
1067 n_samples = _num_samples(X)
1068 column_names = _get_feature_names(X)
1069
1070 if fit_dataframe_and_transform_dataframe:
1071 named_transformers = self.named_transformers_
1072 # check that all names seen in fit are in transform, unless
1073 # they were dropped
1074 non_dropped_indices = [
1075 ind
1076 for name, ind in self._transformer_to_input_indices.items()
1077 if name in named_transformers and named_transformers[name] != "drop"
1078 ]
1079
1080 all_indices = set(chain(*non_dropped_indices))
1081 all_names = set(self.feature_names_in_[ind] for ind in all_indices)
1082
1083 diff = all_names - set(column_names)
1084 if diff:
1085 raise ValueError(f"columns are missing: {diff}")
1086 else:
1087 # ndarray was used for fitting or transforming, thus we only
1088 # check that n_features_in_ is consistent
1089 _check_n_features(self, X, reset=False)
1090
1091 if _routing_enabled():
1092 routed_params = process_routing(self, "transform", **params)
1093 else:
1094 routed_params = self._get_empty_routing()
1095
1096 Xs = self._call_func_on_transformers(
1097 X,
1098 None,
1099 _transform_one,
1100 column_as_labels=fit_dataframe_and_transform_dataframe,
1101 routed_params=routed_params,
1102 )
1103 self._validate_output(Xs)
1104
1105 if not Xs:
1106 # All transformers are None
1107 return np.zeros((n_samples, 0))
1108
1109 return self._hstack(list(Xs), n_samples=n_samples)

1111 def _hstack(self, Xs, *, n_samples):
1112 """Stacks Xs horizontally.
1113
1114 This allows subclasses to control the stacking behavior, while reusing
1115 everything else from ColumnTransformer.
1116
1117 Parameters
1118 ----------
1119 Xs : list of {array-like, sparse matrix, dataframe}
1120 The container to concatenate.
1121 n_samples : int
1122 The number of samples in the input data to checking the transformation
1123 consistency.
1124 """
1125 if self.sparse_output_:
1126 try:
1127 # since all columns should be numeric before stacking them
1128 # in a sparse matrix, `check_array` is used for the
1129 # dtype conversion if necessary.
1130 converted_Xs = [
1131 check_array(X, accept_sparse=True, ensure_all_finite=False)
1132 for X in Xs
1133 ]
1134 except ValueError as e:
1135 raise ValueError(
1136 "For a sparse output, all columns should "
1137 "be a numeric or convertible to a numeric."
1138 ) from e
1139
1140 return sparse.hstack(converted_Xs).tocsr()
1141 else:
1142 Xs = [f.toarray() if sparse.issparse(f) else f for f in Xs]
1143 adapter = _get_container_adapter("transform", self)
1144 if adapter and all(adapter.is_supported_container(X) for X in Xs):
1145 # rename before stacking as it avoids to error on temporary duplicated
1146 # columns
1147 transformer_names = [
1148 t[0]
1149 for t in self._iter(
1150 fitted=True,
1151 column_as_labels=False,
1152 skip_drop=True,
1153 skip_empty_columns=True,
1154 )
1155 ]
1156 feature_names_outs = [X.columns for X in Xs if X.shape[1] != 0]
1157 if self.verbose_feature_names_out:
1158 # `_add_prefix_for_feature_names_out` takes care about raising
1159 # an error if there are duplicated columns.
1160 feature_names_outs = self._add_prefix_for_feature_names_out(
1161 list(zip(transformer_names, feature_names_outs))
1162 )
1163 else:
1164 # check for duplicated columns and raise if any
1165 feature_names_outs = list(chain.from_iterable(feature_names_outs))
1166 feature_names_count = Counter(feature_names_outs)
1167 if any(count > 1 for count in feature_names_count.values()):
1168 duplicated_feature_names = sorted(
1169 name
1170 for name, count in feature_names_count.items()
1171 if count > 1
1172 )
1173 err_msg = (
1174 "Duplicated feature names found before concatenating the"
1175 " outputs of the transformers:"
1176 f" {duplicated_feature_names}.\n"
1177 )
1178 for transformer_name, X in zip(transformer_names, Xs):
1179 if X.shape[1] == 0:
1180 continue
1181 dup_cols_in_transformer = sorted(
1182 set(X.columns).intersection(duplicated_feature_names)
1183 )
1184 if len(dup_cols_in_transformer):
1185 err_msg += (
1186 f"Transformer {transformer_name} has conflicting "
1187 f"columns names: {dup_cols_in_transformer}.\n"
1188 )
1189 raise ValueError(
1190 err_msg
1191 + "Either make sure that the transformers named above "
1192 "do not generate columns with conflicting names or set "
1193 "verbose_feature_names_out=True to automatically "
1194 "prefix to the output feature names with the name "
1195 "of the transformer to prevent any conflicting "
1196 "names."
1197 )
1198
1199 names_idx = 0
1200 for X in Xs:
1201 if X.shape[1] == 0:
1202 continue
1203 names_out = feature_names_outs[names_idx : names_idx + X.shape[1]]
1204 adapter.rename_columns(X, names_out)
1205 names_idx += X.shape[1]
1206
1207 output = adapter.hstack(Xs)
1208 output_samples = output.shape[0]
1209 if output_samples != n_samples:
1210 raise ValueError(
1211 "Concatenating DataFrames from the transformer's output lead to"
1212 " an inconsistent number of samples. The output may have Pandas"
1213 " Indexes that do not match, or that transformers are returning"
1214 " number of samples which are not the same as the number input"
1215 " samples."
1216 )
1217
1218 return output
1219
1220 return np.hstack(Xs)

1371def make_column_transformer(
1372 *transformers,
1373 remainder="drop",
1374 sparse_threshold=0.3,
1375 n_jobs=None,
1376 verbose=False,
1377 verbose_feature_names_out=True,
1378 force_int_remainder_cols="deprecated",
1379):
1380 """Construct a ColumnTransformer from the given transformers.
1381
1382 This is a shorthand for the ColumnTransformer constructor; it does not
1383 require, and does not permit, naming the transformers. Instead, they will
1384 be given names automatically based on their types. It also does not allow
1385 weighting with ``transformer_weights``.
1386
1387 Read more in the :ref:`User Guide <make_column_transformer>`.
1388
1389 Parameters
1390 ----------
1391 *transformers : tuples
1392 Tuples of the form (transformer, columns) specifying the
1393 transformer objects to be applied to subsets of the data.
1394
1395 transformer : {'drop', 'passthrough'} or estimator
1396 Estimator must support :term:`fit` and :term:`transform`.
1397 Special-cased strings 'drop' and 'passthrough' are accepted as
1398 well, to indicate to drop the columns or to pass them through
1399 untransformed, respectively.
1400 columns : str, array-like of str, int, array-like of int, slice, \
1401 array-like of bool or callable
1402 Indexes the data on its second axis. Integers are interpreted as
1403 positional columns, while strings can reference DataFrame columns
1404 by name. A scalar string or int should be used where
1405 ``transformer`` expects X to be a 1d array-like (vector),
1406 otherwise a 2d array will be passed to the transformer.
1407 A callable is passed the input data `X` and can return any of the
1408 above. To select multiple columns by name or dtype, you can use
1409 :obj:`make_column_selector`.
