Detailed Description

Place evenly spaced ticks, with the step size and maximum number of ticks chosen
automatically.

This is a subclass of `~matplotlib.ticker.MaxNLocator`, with parameters
*nbins = 'auto'* and *steps = [1, 2, 2.5, 5, 10]*.
